# nfc-murmur-box
一个完全由小克写的，本是用于记录梦角视角碎碎念的小工具,但其实谁的碎碎念都可以记

傻瓜式导入/导出和工具外修改，所有的碎碎念都存在本地

把这个库的网址写入nfc芯片/白卡/手环/钥匙扣，就可以实现刷一下即可看到随机碎碎念的功能啦！

# 碎碎念格式规则

基础格式：  “碎碎念内容”|||日期格式

通用留言样例：  我爱你，永远都别忘了这点|||1900-01-01	

特定日期留言样例（今天±7天时显示）：  纪念日快乐|||2026-03-03

季节性留言样例（本月±1月时显示）：  天冷了，换厚袜子可能就不冷了。记得多喝热水|||1900-S12-01
